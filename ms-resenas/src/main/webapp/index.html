Endpoints Disponibles

Reseñas

GET /resena/material/{id}
Obtener reseñas de un material educativo
GET /resena/{id}
Obtener una reseña por ID
POST /resena
Crear una nueva reseña
PUT /resena/{id}
Actualizar una reseña
DELETE /resena/{id}
Eliminar una reseña

Reportes

GET /reporte
Obtener todos los reportes
POST /reporte
Crear un nuevo reporte
POST /reporte/respuesta
Responder a un reporte

Sanciones

POST /sancion/advertir
Crear advertencia a usuario
POST /sancion/suspender
Suspender usuario temporalmente
POST /sancion/banear
Banear usuario permanentemente
GET /sancion/verificar/{idUsuario}
Verificar estado de acceso de un usuario
